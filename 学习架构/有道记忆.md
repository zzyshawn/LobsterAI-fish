# LobsterAI 记忆系统

## 一、概述

LobsterAI 的记忆系统允许 AI 记住用户的个人信息、偏好等，在后续对话中自动注入这些记忆，提供个性化服务。

---

## 二、数据库结构

### 2.1 记忆表 (`user_memories`)

**文件**: `src/main/sqliteStore.ts:121`

```sql
CREATE TABLE IF NOT EXISTS user_memories (
  id TEXT PRIMARY KEY,
  text TEXT NOT NULL,                    -- 记忆内容
  fingerprint TEXT NOT NULL,             -- 内容指纹（用于去重）
  confidence REAL NOT NULL DEFAULT 0.75, -- 置信度 (0-1)
  is_explicit INTEGER NOT NULL DEFAULT 0,-- 是否显式记忆
  status TEXT NOT NULL DEFAULT 'created',-- 状态: created/stale/deleted
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL,
  last_used_at INTEGER                   -- 最后使用时间
);
```

### 2.2 记忆来源表 (`user_memory_sources`)

```sql
CREATE TABLE IF NOT EXISTS user_memory_sources (
  id TEXT PRIMARY KEY,
  memory_id TEXT NOT NULL,               -- 关联记忆 ID
  session_id TEXT,                       -- 来源会话
  message_id TEXT,                       -- 来源消息
  role TEXT NOT NULL DEFAULT 'system',   -- 角色
  is_active INTEGER NOT NULL DEFAULT 1,
  created_at INTEGER NOT NULL,
  FOREIGN KEY (memory_id) REFERENCES user_memories(id) ON DELETE CASCADE
);
```

## 三、记忆提取机制

```typescript
export function extractTurnMemoryChanges(options: ExtractTurnMemoryOptions): ExtractedMemoryChange[] {
  // 1. 提取显式添加
  const explicitAdds = extractExplicit(userText, 'add', EXPLICIT_ADD_RE, 'explicit:add-command');
  // 2. 提取显式删除
  const explicitDeletes = extractExplicit(userText, 'delete', EXPLICIT_DELETE_RE, 'explicit:delete-command');
  // 3. 提取隐式记忆
  const implicitAdds = extractImplicit(options);
  
  return [...explicitDeletes, ...explicitAdds, ...implicitAdds];
}
```

### 3.4 过滤规则

**不记录的内容**:
- 问答句 (`isQuestionLikeMemoryText`)
- 闲聊 (`ok/thanks/好的/收到`)
- 临时信息 (`今天/昨天/新闻`)
- 代码块内容
- 命令执行相关

```typescript
private buildUserMemoriesXml(): string {
  if (!config.memoryEnabled) {
    return '<userMemories></userMemories>';
  }

  const memories = this.store.listUserMemories({
    status: 'created',
    limit: config.memoryUserMemoriesMaxItems,
  });

  const lines = memories.map((memory) => `- ${this.escapeXml(memory.text)}`);
  return `<userMemories>\n${lines.join('\n')}\n</userMemories>`;
}
```

## 六、记忆更新流程

### 6.1 会话结束时触发

**文件**: `coworkRunner.ts:396`

```typescript
private applyTurnMemoryUpdatesForSession(sessionId: string): void {
  if (!config.memoryEnabled) return;

  // 获取最后一轮对话
  const lastUser = [...session.messages].reverse().find(m => m.type === 'user');
  const lastAssistant = [...session.messages].reverse().find(m => m.type === 'assistant');

  // 加入异步队列
  this.turnMemoryQueue.push({
    sessionId,
    userText: lastUser.content,
    assistantText: lastAssistant.content,
    implicitEnabled: config.memoryImplicitUpdateEnabled,
    guardLevel: config.memoryGuardLevel,
  });

  void this.drainTurnMemoryQueue();
}
```

### 6.2 异步处理队列

**文件**: `coworkRunner.ts:456`

```typescript
private async drainTurnMemoryQueue(): Promise<void> {
  while (this.turnMemoryQueue.length > 0) {
    const job = this.turnMemoryQueue.shift();
    const result = await this.store.applyTurnMemoryUpdates({
      sessionId: job.sessionId,
      userText: job.userText,
      assistantText: job.assistantText,
      implicitEnabled: job.implicitEnabled,
      guardLevel: job.guardLevel,
    });
  }
}
```
