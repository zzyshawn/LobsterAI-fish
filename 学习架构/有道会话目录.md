# LobsterAI 会话目录建立过程

## 一、概述

会话目录（工作目录）是 Agent 执行任务的根目录，决定了文件操作的范围和沙箱隔离的边界。

### 3.2 目录解析过程

**方法**: `resolveTaskWorkingDirectory()`

**文件**: `src/main/main.ts:188`

```typescript
const resolveTaskWorkingDirectory = (workspaceRoot: string): string => {
  const resolvedWorkspaceRoot = path.resolve(workspaceRoot);
  
  // 自动创建目录（如果不存在）
  fs.mkdirSync(resolvedWorkspaceRoot, { recursive: true });
  
  // 验证是目录
  if (!fs.statSync(resolvedWorkspaceRoot).isDirectory()) {
    throw new Error(`Selected workspace is not a directory: ${resolvedWorkspaceRoot}`);
  }
  
  return resolvedWorkspaceRoot;
};
```

## 四、会话数据结构

### 4.1 数据库表结构

**表名**: `cowork_sessions`

**文件**: `src/main/sqliteStore.ts:95`

```sql
CREATE TABLE IF NOT EXISTS cowork_sessions (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  claude_session_id TEXT,
  status TEXT NOT NULL DEFAULT 'idle',
  cwd TEXT NOT NULL,                    -- 工作目录
  system_prompt TEXT,
  execution_mode TEXT,
  active_skill_ids TEXT,
  pinned INTEGER NOT NULL DEFAULT 0,
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL
);
```

### 4.2 创建会话记录

**方法**: `CoworkStore.createSession()`

**文件**: `src/main/coworkStore.ts:508`

```typescript
createSession(
  title: string,
  cwd: string,
  systemPrompt: string = '',
  executionMode: CoworkExecutionMode = 'local',
  activeSkillIds: string[] = []
): CoworkSession {
  const id = uuidv4();
  const now = Date.now();

  this.db.run(`
    INSERT INTO cowork_sessions (
      id, title, claude_session_id, status, cwd, system_prompt,
      execution_mode, active_skill_ids, pinned, created_at, updated_at
    ) VALUES (?, ?, NULL, 'idle', ?, ?, ?, ?, 0, ?, ?)
  `, [id, title, cwd, systemPrompt, executionMode, JSON.stringify(activeSkillIds), now, now]);

  return { id, title, cwd, status: 'idle', ... };
}
```

## 十、常见问题

### Q1: 目录不存在会怎样？

**答**: `resolveTaskWorkingDirectory` 会自动创建（`recursive: true`）

### Q2: 目录权限不足会怎样？

**答**: 抛出异常，会话创建失败，返回错误信息

### Q3: 如何获取当前工作目录？

```typescript
// 前端
const cwd = config.workingDirectory;

// 后端
const config = coworkStore.getConfig();
const cwd = config.workingDirectory;
```

### Q4: 多个会话可以共用同一目录吗？

**答**: 可以，每个会话独立存储，目录只是工作空间
